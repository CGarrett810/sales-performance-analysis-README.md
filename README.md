# 📊 Excel Sales Analysis Project

## 📌 Overview
This project analyzes sales data using Microsoft Excel to identify trends in regions, product categories, and time-based performance.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## 📈 Key Insights
- East region shows the highest sales performance  
- Furniture category generates the most profit  
- Sales over a short period, these variations highlight changes in customer demand and can inform short-term forecasting and operational planning

## 📊 Dashboard
The dashboard visualizes key performance metrics, including regional sales, category profit, and sales trends over time.
